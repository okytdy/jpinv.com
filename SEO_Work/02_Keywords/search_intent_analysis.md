## 2. 主要 KW の検索意図マップ

### 2.1 「海外IR」(High volume, broad)
- **顕在意図**: 海外投資家へのIR活動の進め方を知りたい
- **潜在意図**: 既に何かしらやっているが効果に確信がない / 上司に説明する材料がほしい
- **コンテンツ要件**:
  - 「海外IRとは」の定義は不要 (既に知っている)
  - 「他社はどうしているか」のベンチマーク欲求
  - 「何を測れば良いか」の指標感
  - 「自社の現状を診断する手段」へのリーチ
- **LP対応**: `/サービス/diagnostic/` で「現状診断 → 改善案」の流れに引き込む

### 2.2 「英文IR」「英文IRレビュー」
- **顕在意図**: 英文IR資料の品質を確かめたい / 改善したい
- **潜在意図**: 自社の英訳が投資家に届いているか確証がない / 既存翻訳会社のアウトプットに違和感がある
- **コンテンツ要件**:
  - 「翻訳の品質」ではなく「投資家に届くか」の評価軸
  - 「ネイティブチェック」ではない別レイヤーの価値
  - 具体的にレビューする観点 (numbers, hedging, KPI alignment, capital allocation language)
- **LP対応**: `/サービス/english-review/` + 記事 `/articles/why-correct-english-doesnt-reach/`

### 2.3 「英文開示」
- **顕在意図**: 適時開示・統合報告書・コーポレートガバナンス報告書の英文をどう作るか / レビューするか
- **潜在意図**: 制度開示の英文化要件への対応負担 / 投資家からのフィードバックが薄くて改善方法が不明
- **コンテンツ要件**:
  - 「制度遵守」レベルではない、投資家視点でのレビュー観点
  - 各開示書類タイプ別の落とし穴
  - 翻訳会社の納品物との違い
- **LP対応**: `/サービス/disclosure-review/`

### 2.4 「海外投資家 面談」「機関投資家 面談 通訳」
- **顕在意図**: 海外投資家との面談をどう準備するか / 良い通訳者を見つけたい
- **潜在意図**: 過去の面談で何が伝わっていたか分からない / 通訳の出来で結果が大きく変わる感覚
- **コンテンツ要件**:
  - 面談前後で何を整理すべきか
  - 「IR現場経験のある通訳者」の価値
  - 面談観察+フィードバックレポートの提供
- **LP対応**: `/サービス/meeting-support/` + `/サービス/interpretation/` + 記事 `/articles/first-10-minutes-overseas-investor/`

### 2.5 「決算説明会 英語」「決算説明会 通訳」
- **顕在意図**: 決算説明会の英語対応をどうするか
- **潜在意図**: 同時通訳の質に懸念 / 英文スクリプトと日本語実演の温度差
- **コンテンツ要件**:
  - スクリプト, QA, 通訳, ポストイベント分析を一体で
  - 上場企業の決算実務へのフィット感
- **LP対応**: `/サービス/earnings-review/` + `/サービス/interpretation/`

### 2.6 「IR英訳」「決算短信 英訳」「適時開示 英訳」
- **顕在意図**: 翻訳サービスを探している (= 翻訳会社志向)
- **潜在意図**: 翻訳の質に不満を感じている可能性が高い
- **戦略**: ここに**そのまま「翻訳します」と書くと格安翻訳会社と価格競争**になる。**「翻訳の前段階で、何を伝えるかを整理する」**という上位概念に引き上げる。
- **コンテンツ要件**:
  - 翻訳との違いを明確化
  - 「翻訳依頼の前に投資家視点で整理する」価値
- **LP対応**: `/サービス/english-review/` (翻訳との違いを冒頭で示す) + `/サービス/disclosure-review/`

### 2.7 long-tail / authority KW
- 「英文IR 伝わらない」「海外投資家 IR 10分」「英文IR 誤解」など
- **意図**: 既に何度か海外IR現場を経験した中堅以上のIR担当が、自分の疑問を言語化して検索
- **コンテンツ要件**: 実務観察ベース、匿名化された事例、断定的すぎず誠実な分析
- **記事対応**: A1〜A6 の authority article
