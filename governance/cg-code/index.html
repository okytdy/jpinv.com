テーマ 2 · コーポレートガバナンス・コード · 2015年 → 2021年

会社と投資家の対話ルールを押さえる

コーポレートガバナンス・コード、CG報告書、取締役会、サステナビリティ、多様性など、対話の土台を整理する。

2015年コードの構造、2018年改訂で入った資本コスト、2021年改訂で強まったプライム市場・サステナビリティ・多様性の要請を、CG報告書の読み方と合わせて確認します。

このテーマで分かること

2015年コードの構造、2018年改訂で入った資本コスト、2021年改訂で強まったプライム市場・サステナビリティ・多様性の要請を、CG報告書の読み方と合わせて確認します。

IR実務で問題になる場面

投資家面談で「コードのどの原則に基づく説明か」「CG報告書のどこに書いているか」を聞かれたとき、法務・総務・IRが同じ前提で答える必要がある場面。

記事一覧

記事 2.1 2015年コーポレートガバナンス・コード ― コンプライ・オア・エクスプレインの基礎 2015年コードの要点は、5つの基本原則、30の原則、38の補充原則という構造と、コンプライ・オア・エクスプレインによるソフトロー運用にある。 この記事を読む → 記事 2.2 2018年改訂 ― 資本効率がコードに入った瞬間 2018年改訂では資本コスト、政策保有株式、CEO後継者計画、企業年金のアセットオーナー機能が前面に出た。 この記事を読む → 記事 2.3 2021年改訂 ― プライム市場・サステナビリティ・多様性 2021年改訂は、2022年4月の市場区分見直しを背景に、プライム市場上場会社への独立社外取締役、スキル・マトリックス、TCFD相当の気候開示、英文開示、電子議決権行使などを強めた。 この記事を読む → 記事 2.4 コーポレートガバナンス報告書を15分で読む方法 コーポレートガバナンス報告書は、コードへの対応状況、取締役会構成、政策保有株式、関連当事者、株主・ステークホルダー対応を短時間で読むための最重要資料である。 この記事を読む → 記事 2.5 コード vs ガイドライン ― 経済産業省の体系 経済産業省のCGSガイドライン、公正なM&Aの在り方に関する指針、グループ・ガバナンス実務指針などは、コードを実務に落とすための詳細な補助線として機能する。 この記事を読む →

まず読むなら

まずは2.1でコードの仕組みを確認し、2.4でCG報告書の読み方を押さえると、改訂論点を実務に落とし込みやすくなります。
