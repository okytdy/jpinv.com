IR実務で読むガバナンス改革

PBR、資本コスト、英文開示、政策保有株。IRで聞かれる論点を、背景から整理する。

PBR1倍割れ、資本コスト、英文開示、政策保有株、市場区分の見直し。投資家との面談で出やすい論点を、制度改正の流れと実務上の意味に分けて整理します。

必要な論点から読み始められるよう、テーマごとに記事を並べています。社内説明、投資家面談、CG報告書や統合報告書の確認に使う前提知識として参照してください。

テーマ 1 · 基礎 ― 改革の起点 · 1990年代 → 2014年

制度改革はなぜ始まったのか

ROE8%、伊藤レポート、アベノミクス、スチュワードシップ・コードまで、現在の議論の前提を押さえる。

このテーマで分かること

日本企業の資本効率がなぜ投資家の主要論点になったのか、メインバンク・政策保有株・株主軽視といった改革前の構造が、2014年以降の制度改革へどうつながったのかを整理します。

テーマ概要を見る →

記事 1.1 「8%」が日本を変えた理由 1989年12月29日のNikkei 225 38,915.87、TOPIX 2,881.37から、2024年に至る長い停滞を、景気循環ではなく資本効率とガバナンスの問題として捉え直す。伊藤レポートが提示したROE8%という基準は、投資家が想定する資本コストを上回るための最低線として、以後の政策議論の共通言語になった。 この記事を読む → 記事 1.2 メインバンクから水野氏まで ― 改革前の日本のガバナンス構造 メインバンク、系列、政策保有株式、社内昇進中心の取締役会という改革前の構造を、単なる慣行ではなく資本市場からの規律を弱める制度的な組み合わせとして読む。 この記事を読む → 記事 1.3 アベノミクスの第三の矢 ― ガバナンスが成長政策になった経緯 アベノミクスの第三の矢は、金融緩和や財政政策だけではなく、企業の稼ぐ力と資本配分を変える成長戦略としてガバナンス改革を位置づけた。 この記事を読む → 記事 1.4 車の両輪 ― スチュワードシップ・コード(2014/2017/2020/2025) スチュワードシップ・コードは、上場会社だけでなく機関投資家にも説明責任を求め、企業統治改革を「会社側」と「投資家側」の両輪で動かす仕組みにした。 この記事を読む →

記事 2.1 2015年コーポレートガバナンス・コード ― コンプライ・オア・エクスプレインの基礎 2015年コードの要点は、5つの基本原則、30の原則、38の補充原則という構造と、コンプライ・オア・エクスプレインによるソフトロー運用にある。 この記事を読む → 記事 2.2 2018年改訂 ― 資本効率がコードに入った瞬間 2018年改訂では資本コスト、政策保有株式、CEO後継者計画、企業年金のアセットオーナー機能が前面に出た。 この記事を読む → 記事 2.3 2021年改訂 ― プライム市場・サステナビリティ・多様性 2021年改訂は、2022年4月の市場区分見直しを背景に、プライム市場上場会社への独立社外取締役、スキル・マトリックス、TCFD相当の気候開示、英文開示、電子議決権行使などを強めた。 この記事を読む → 記事 2.4 コーポレートガバナンス報告書を15分で読む方法 コーポレートガバナンス報告書は、コードへの対応状況、取締役会構成、政策保有株式、関連当事者、株主・ステークホルダー対応を短時間で読むための最重要資料である。 この記事を読む → 記事 2.5 コード vs ガイドライン ― 経済産業省の体系 経済産業省のCGSガイドライン、公正なM&Aの在り方に関する指針、グループ・ガバナンス実務指針などは、コードを実務に落とすための詳細な補助線として機能する。 この記事を読む →
テーマ 3 · 市場区分の見直し · 2022年 → 2025年

市場区分と上場維持基準を理解する

プライム・スタンダード・グロース、市場再編、流通株式比率、TOPIX見直しをIR実務の観点で読む。

このテーマで分かること

市場再編の狙い、プライム・スタンダード・グロースの違い、流通株式比率、経過措置の終了、TOPIX見直しが企業行動と投資家説明に与える影響を整理します。

テーマ概要を見る →

記事 3.1 4区分から3区分へ ― 2022年4月の市場再編 2022年4月4日の市場再編は、市場第一部・第二部・マザーズ・JASDAQという4区分を、プライム、スタンダード、グロースの3市場へ整理した。 この記事を読む → 記事 3.2 流通株式比率 ― 政策保有株を動かした一つの数値 流通株式比率は、固定的な大株主や政策保有株式を除いた実質的な市場流通性を測る指標であり、上場維持基準の中でも行動変容を促す力が大きい。 この記事を読む → 記事 3.3 2025年3月の崖 ― 経過措置の終了と改善期間 2025年3月の経過措置終了は、形式的な先送りを認めず、基準未達企業に改善期間と上場廃止リスクを具体的に意識させる転換点となった。 この記事を読む → 記事 3.4 TOPIX 2.0 ― 全銘柄インデックスから選別インデックスへ TOPIX改革は、市場区分改革と並ぶ構造改革であり、単なる全上場銘柄指数から、流動性と市場評価を反映する選別的なユニバースへ移行する。 この記事を読む → 記事 3.5 グロース市場2030年の崖 ― 100億円基準の意味 グロース市場改革では、2030年を見据えて時価総額100億円基準が焦点となり、上場後も成長を継続できる市場設計が問われている。 この記事を読む →
テーマ 4 · 資本効率と企業価値 · 2023年3月 → 現在

資本コスト・PBR・ROEを説明する

東証の資本コスト要請、PBR1倍割れ、ROIC、WACC、エクイティ・スプレッド、開示改善を整理する。

このテーマで分かること

2023年3月の東証要請を起点に、PBR1倍割れ、資本コスト、ROE・ROIC、WACC、エクイティ・スプレッド、開示企業リスト、投資家目線とのずれを実務の言葉で整理します。

テーマ概要を見る →

記事 4.1 PBR1倍割れは目標ではなく、市場の評決 2023年3月の東証要請は、PBR1倍割れを目標ではなく市場からの評価として捉え、資本コストや株価を意識した経営の実行を求めた。 この記事を読む → 記事 4.2 WACC・ROIC・エクイティ・スプレッド ― IR担当が押さえるべき新語 WACC、ROIC、ROE、エクイティ・スプレッド、残余利益といった語彙は、投資家との対話だけでなく、社内の資本配分を説明するための実務言語である。 この記事を読む → 記事 4.3 開示企業リスト ― 公表を通じた外部規律と優良事例の可視化 東証の開示企業リストは、罰則ではなく公表を通じて参加状況を可視化し、外部規律と優良事例の横展開を同時に進める仕組みである。 この記事を読む → 記事 4.4 資本コスト開示の7つの落とし穴 ― 「投資家視点との乖離」事例集 東証の「投資者の視点を踏まえた対応のポイントと事例」は、良い事例だけでなく、投資家目線とずれる典型的な開示を明示している点に実務上の価値がある。 この記事を読む → 記事 4.5 開示から実行へ ― 2026年4月「アップデート」が変えたもの 2026年4月のアップデートは、開示の有無から、取締役会での議論、施策の実行、進捗検証、投資家との対話へ焦点を移した。 この記事を読む →
テーマ 5 · 現在の主要論点 · 2024年 → 2027年

これからのガバナンス実務に備える

英文開示、政策保有株、上場子会社、買収対応、SSBJ、取締役会多様性、アクティビズムを確認する。

このテーマで分かること

英文開示、政策保有株、上場子会社、買収対応、サステナビリティ基準、取締役会多様性、アクティビズム、政策形成の場を、今後のIR実務に関係する論点として整理します。

テーマ概要を見る →

記事 5.1 2025年4月は通過点、終着点ではない ― 英文開示義務化 2025年4月からの英文開示義務化は終着点ではなく、プライム市場上場会社が海外投資家に同時・同質の情報を届ける運用開始の節目である。 この記事を読む → 記事 5.2 政策保有株のエンドゲーム ― 「個別検証」開示の時代 政策保有株式を総額だけで説明する時代は終わり、個別銘柄ごとの保有合理性、縮減方針、資本効率への影響を説明する段階に入った。 この記事を読む → 記事 5.3 豊田自動織機ディールが終わらせた系列構造 上場子会社をめぐる議論は、親子上場そのものの是非だけでなく、少数株主保護、利益相反、グループ資本配分の説明責任を問うものになっている。 この記事を読む → 記事 5.4 2023年の買収行動指針は、敵対的買収を例外視せず、企業価値と株主共同の利益を基準に、取締役会の対応を透明化する枠組みを示した。 この記事を読む → 記事 5.5 SSBJ実務 ― 3段階のサステナビリティ開示ロードマップ SSBJ基準は、サステナビリティ情報を任意のCSR説明ではなく、投資判断に必要な制度開示へ移行させるロードマップである。 この記事を読む → 記事 5.6 「30 by 30」はパイプライン問題である 女性役員比率30%という目標は、任命時点の数字だけでなく、候補者育成、執行経験、指名プロセスを含むパイプラインの問題として捉える必要がある。 この記事を読む → 記事 5.7 スチュワードシップ・コード3.0以後の協働エンゲージメント 2025年のスチュワードシップ・コード改訂と大量保有報告制度の見直しは、協働エンゲージメントを市場の通常機能として位置づけ直した。 この記事を読む → 記事 5.8 次のガバナンス・ルールを書いているワーキング・グループ群 金融庁、東証、経済産業省、法務省、SSBJなどの審議会・研究会は、次の制度変更を先取りするための一次情報源である。 この記事を読む →
